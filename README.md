# 📝 ToDo List App

A simple and responsive ToDo List web application built with **HTML, Tailwind CSS, and JavaScript**.
It allows users to add tasks and organize them by days of the week.

---

## 🚀 Features

* Add tasks easily
* Assign tasks to specific days (Monday–Sunday)
* Tasks are displayed in a weekly table format
* Clean and responsive UI using Tailwind CSS
* Simple and beginner-friendly JavaScript logic

---

## 🛠️ Built With

* HTML5
* Tailwind CSS
* Vanilla JavaScript

## ⚙️ How to Run the Project

1. Clone the repository:

```bash
git clone https://github.com/your-username/todo-list-app.git
```



2. Open `index.html` in your browser.

---

## ✨ How It Works

1. Type your task in the input field
2. Select the day of the week
3. Click **Add Task**
4. The task appears under the selected day in the table
